User Registration

POST /api/v1/auth/users/register/

Register a new user to the BookStore User DB.

Query Parameters

  • string – email (required) – E-mail of registering user

  • string – username (required) – Username of registering user

  • string – password (required) – Password of registering user

Request Headers

  • Authorization – token

Example response:
