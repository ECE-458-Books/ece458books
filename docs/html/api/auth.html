User Registration

POST /api/v1/auth/users/register/

Register a new user to the BookStore User DB.

Query Parameters:

  • email (string) – email of registering user

  • username (string) – username of registering user

  • password (string) – password of registering user

Example request

$ curl --location --request POST 'https://books.colab.duke.edu:8000/api/v1/auth/users/register/' \
--header 'Content-Type: application/json' \
--data-raw '{
    "email": "<email>",
    "username": "<username>",
    "password": "<password>"
}'

import requests
import json

url = "https://books.colab.duke.edu:8000/api/v1/auth/users/register/"

payload = json.dumps({
    "email": "<email>",
    "username": "<username>",
    "password": "<password>"
})

headers = {
'Content-Type': 'application/json'
}

response = requests.request("POST", url, headers=headers, data=payload)

print(response.text)

Example response

{
    "email": "<email>",
    "username": "<username>"
}
